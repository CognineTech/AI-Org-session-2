The Road to Deep Learning

Rules  →  Pattern Learning  →  Automatic Intelligence

Evolution
Phase 01
Rule-Based Systems
Humans write every rule by hand
 Rules become obsolete
Phase 02
Machine Learning
Learns patterns from labeled examples
 Needs manual feature engineering
Phase 03
Deep Learning
Learns features directly from raw data
 Automatic representation learning

The Key Insight: "Why manually decide what patterns matter?" — Deep Learning answers this by learning everything automatically from raw data.
